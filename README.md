# Spectator Engineering Training - Spec Sources Project

<p>After git cloning the repository, follow these steps for setup:</p>

### Backend FastAPI
<ul>
    <li>cd into app/ and install all necessary packages</li>
    <li>create a virtual environment using python3 -m venv venv, and then run source venv/bin/activate to activate the virtual environment</li>
    <li>run pip install -r requirements.txt to install all packages</li>
    <li>run fastapi dev main.py to run the backend</li>
</ul>

### Frontend React App
<ul>
    <li>cd into specsourcesv2/ and run npm start</li>
    <li>the app should be running on <a>http://localhost:3000</a>. visit it in the browser to use the app</li>
</ul>
